# homebrew-tap

Homebrew formulae for [@salvodicara](https://github.com/salvodicara)'s tools.

## Install

```sh
brew install salvodicara/tap/git-recap
```

or tap once, then install by name:

```sh
brew tap salvodicara/tap
brew install git-recap
```

## Formulae

- **git-recap** — reconstruct a work journal from your git history
  ([repo](https://github.com/salvodicara/git-recap)).
